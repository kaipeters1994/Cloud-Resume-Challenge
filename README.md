This is the Cloud Resume Challenge (AWS Version)

This project's purpose is to demonstrate cloud, devops, and serverless skils. This project aims to 
demonstrate a personal resume website hosted on AWS with full infrastructure automation, serverless compute
and analytics tracking with a live counter. 

STEPS:
1. Have an AWS Certification
2. Format your resume in HTML
3. Styled with CSS
4. Create static website via S3
OPTIONAL: Hosted custom domain with Route 53
5. Enable HTTPS
6. Configure DNS
7. Add a views tracker with Javascript
8. Create database
9. API Gateway
10. Python code in Lambda
11. Run tests against python code
12. IaC version
13. Github Repository
14. CI/CD Front End
15. CI/CD Back End
16. Documentation

TECHNOLOGIES USED:
-AWS S3
-AWS CloudFront
